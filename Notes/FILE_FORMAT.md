# File Formats

## 1. MoCap Datest Types

| Format    | Content Type                   | Typical Extension |
| --------- | ------------------------------ | ----------------- |
| BVH       | Hierarchical skeleton + motion | `.bvh`            |
| FBX       | Mesh + rig + animation         | `.fbx`            |
| C3D       | Marker-based mocap             | `.c3d`            |
| CSV / TXT | Raw 3D joint coordinates       | `.csv`, `.txt`    |
| NPZ / MAT | Structured arrays              | `.npz`, `.mat`    |
